Connecting with Yabe (Yet Another BACnet Explorer)

You can connect to the Campus Simulator using Yabe to explore BACnet objects, read values, and write overrides.

Prerequisites

  • Download and install Yabe on your Windows machine.
  • Ensure you have network access to the simulator host.

Connection Steps

  1. Open Yabe.
  2. Click the Add device button (green plus icon).
  3. In the "Add device" dialog:
    • Port: 47808 (Default)
    • Local IP: Select your network interface that can reach the simulator.
  4. Click Start.
  5. Yabe will send a Who-Is broadcast. The simulator should respond as:
    Device ID: 9999
    Name: CampusGateway
    Vendor: 15
  6. Click on the device to see the list of objects (Analog Values, Binary Values, etc.).

Troubleshooting

If you cannot discover the device:

  • Ensure UDP port 47808 is open on the firewall.
  • If running in Docker, ensure the port is mapped correctly (it is by default).
  • Try adding the simulator IP explicitly if broadcast is blocked (Yabe -> Add device -> Remote BBMD -> Enter Simulator IP:47808).

Connecting with BACnet/SC (Secure Connect)

The simulator provides a BACnet/SC Hub endpoint accessible via WebSocket (WSS).

Update: The simulator now supports standard binary BACnet/SC connections! You can use Yabe or other standard clients.

Connection Details

Hub URL: wss://sim.hill.coffee/bacnet-sc
Port: 443 (HTTPS/WSS)
Authentication: None (Open Hub)

Yabe Configuration

  1. Open Yabe.
  2. Click Add device.
  3. Select BACnet/SC tab.
  4. Enter the Hub URL: wss://sim.hill.coffee/bacnet-sc
  5. Connect.

Modbus TCP

The simulator exposes a Modbus TCP server for integration with industrial control systems.

Connection Details

Host: sim.hill.coffee (or IP)
Port: 5020
Unit ID: 1 (Gateway)

Register Mapping

Points are mapped to Holding Registers (4xxxx) and Input Registers (3xxxx).

  • Campus OAT: Register 0 (Input)
  • Electrical Main kW: Register 100 (Input)
  • Building Points: Dynamically assigned starting at 1000.

Note: Values are typically scaled by 100 (e.g., 72.5°F = 7250).
